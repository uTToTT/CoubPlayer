Кладите сюда свои иконки — плеер подхватит их автоматически,
без правок HTML/CSS/JS. Формат: SVG (предпочтительно, одноцветный,
использует currentColor) или PNG/WEBP на прозрачном фоне, квадрат.

Имя файла = имя иконки. Поддерживаемые имена:

  prev.svg              — стрелка "предыдущее видео"
  next.svg              — стрелка "следующее видео"
  fullscreen.svg        — полноэкранный режим
  restart.svg           — перезапустить видео
  copy-link.svg         — скопировать ссылку
  tags.svg              — теги видео
  playlist-add.svg      — добавить в плейлист
  folder.svg            — открыть папку с файлами
  playlist.svg          — иконка плейлиста (в шапке и во вкладках)
  download.svg          — загрузить видео
  link.svg              — добавить по ссылке
  sync-liked.svg        — синхронизировать Liked
  sync-bookmarks.svg    — синхронизировать Bookmarks
  add.svg               — создать новый плейлист / тег

Пример: чтобы заменить иконку перезапуска — сохраните файл как
icons/restart.svg (или .png/.webp) и обновите страницу.
